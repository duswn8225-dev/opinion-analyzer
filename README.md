# 📊 주관식 의견 분석 도구 (Opinion Analyzer)

**강의평가, 교육만족도 조사** 등 교육 관련 주관식 의견을 **웹에서 바로 분석**할 수 있는 올인원 도구입니다.

## ✨ 주요 기능

- 📁 **CSV/Excel 업로드**: 강의평가 데이터 일괄 업로드
- 🔍 **다차원 분석**: 과정별, 강의일자, 강사명, 교과목명, 문항별 분석
- 🔄 **키워드 기반 자동 묶기**: 비슷한 의견끼리 자동 분류
- 😊 **감정 분석**: 긍정/부정/중립 감정 경향 분석
- 📝 **핵심 요약**: 각 그룹별 의견 요약
- ⭐ **대표 의견**: 그룹을 대표하는 문장 추출
- 📊 **상세 리포트**: 다양한 필터링 및 비교 분석
- 📥 **결과 내보내기**: Excel, JSON 형식으로 저장

## 🚀 시작하기

### 설치

```bash
# 저장소 클론
git clone git@github.com:duswn8225-dev/opinion-analyzer.git
cd opinion-analyzer

# Python 패키지 설치
pip install -r requirements.txt
```

### 실행

```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 로 자동 실행됩니다!

## 📋 CSV 파일 양식

### 필수 컬럼:

| 컬럼명 | 설명 | 예시 | 필수여부 |
|--------|------|------|--------|
| `id` | 응답자 ID | 001, 002 | ✅ 필수 |
| `course` | 과정명 | 신입사원교육, 리더십과정 | ✅ 필수 |
| `lecture_date` | 강의일자 | 2024-01-15 | ✅ 필수 |
| `instructor` | 강사명 | 김강사, 이강사 | ✅ 필수 |
| `subject` | 교과목명 | 통신 기초, 고객응대 | ✅ 필수 |
| `question_num` | 문항번호 | 1, 2, 3 | ✅ 필수 |
| `question_text` | 질문 내용 | 강의가 도움이 되었나요? | ✅ 필수 |
| `answer` | 주관식 답변 | 매우 도움이 되었습니다 | ✅ 필수 |

### 샘플 데이터 (sample_data.csv):

```csv
id,course,lecture_date,instructor,subject,question_num,question_text,answer
001,신입사원교육,2024-01-15,김강사,통신 기초,1,강의가 도움이 되었나요?,강의 내용이 체계적이고 이해하기 쉬웠어요
002,신입사원교육,2024-01-15,김강사,통신 기초,1,강의가 도움이 되었나요?,실무에 바로 적용할 수 있는 내용이 좋았어요
003,신입사원교육,2024-01-15,김강사,통신 기초,2,강사의 강의 방식은 만족하나요?,강사님이 학생 반응을 잘 봐주셨어요
004,리더십과정,2024-01-16,이강사,고객응대,1,강의가 도움이 되었나요?,너무 빨리 진행되어 따라가기 어려웠어요
005,리더십과정,2024-01-16,이강사,고객응대,2,강사의 강의 방식은 만족하나요?,사례 중심으로 설명해주셔서 이해가 잘 됐어요
```

## 🎯 사용 방법

### 1️⃣ 파일 준비
- CSV 파일 양식에 맞게 데이터 준비
- 또는 제공된 `sample_data.csv` 템플릿 사용

### 2️⃣ 파일 업로드
- 웹에서 "파일 업로드" 버튼으로 CSV/Excel 파일 선택

### 3️⃣ 분석 필터 설정
- **과정별**: 특정 과정만 분석
- **강의일자**: 기간별 분석
- **강사명**: 강사별 의견 비교
- **교과목명**: 과목별 의견 추출
- **문항별**: 특정 문항의 답변 분석

### 4️⃣ 분석 시작
- "분석 시작" 버튼 클릭

### 5️⃣ 결과 확인
- 키워드 기반 의견 그룹 확인
- 감정 경향 시각화 (차트)
- 각 그룹의 핵심 요약 및 대표 의견
- 필터별 상세 통계

### 6️⃣ 결과 내보내기
- Excel: 그룹화된 의견, 감정, 요약 정보
- JSON: 구조화된 데이터
- PDF: 최종 리포트

## 🛠 기술 스택

- **Frontend/UI**: Streamlit
- **Backend**: Python
- **NLP**: KoBERT, Scikit-learn
- **데이터처리**: Pandas, NumPy
- **시각화**: Plotly, Matplotlib, Seaborn
- **파일처리**: openpyxl (Excel)

## 📊 분석 로직

### 1. 데이터 전처리
- 텍스트 정규화 (띄어쓰기, 문장부호 정리)
- 불용어 제거 (한국어)
- 형태소 분석

### 2. 의견 그룹핑
- TF-IDF 기반 유사도 계산
- K-means 클러스터링
- 같은 의미의 답변 자동 분류

### 3. 감정 분석
- KoBERT 모델 기반 감정 분류
- 긍정/부정/중립 분포 계산

### 4. 요약 & 대표문장 추출
- 각 클러스터의 핵심 키워드 추출
- 대표성 높은 문장 자동 선택

### 5. 다차원 필터링
- 과정/강사/교과목/문항별 세부 분석
- 조건별 비교 분석
